# Extraction Patterns

Use this reference when source material is large or mixed.

## Search Plan

Start with 3-8 keywords:

- core topic names
- product names
- metric names
- audience terms
- date/version terms
- synonyms

Run several narrow searches instead of one broad full read.

## Evidence Card

```text
Claim:
Evidence:
Source:
Location:
Reliability:
Deck use:
Visual suggestion:
```

## Spreadsheet Preview

For spreadsheets, capture:

- sheet name
- row count if available
- column names
- likely metric columns
- date columns
- categorical dimensions
- missing values or caveats

## PDF / DOCX Preview

For documents, capture:

- title
- section headings
- executive summary claims
- tables / figures
- definitions
- conclusion / recommendations
